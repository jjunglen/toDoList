# Todo List App

A simple todo list built with HTML, CSS, and JavaScript.

## What it does

- Add todo items to a list
- Check them off when done
- Delete them when you no longer need them
- Saves your todos so they're still there when you refresh the page

## How to open it

1. Download or clone this project to your computer
2. Open the folder
3. Double-click `index.html` — it will open in your browser

That's it! No installs needed.

## Files

- `index.html` — the page structure
- `style.css` — the styles
- `script.js` — all the JavaScript logic
